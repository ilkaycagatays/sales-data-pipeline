# 🚀 Sales Data Pipeline

## 📌 Overview
End-to-end data pipeline that extracts raw sales data, 
transforms and cleans it using Python, loads it into a 
SQLite database, and visualizes KPIs in a dashboard.

## 🏗️ Architecture
```
Raw CSV → ETL Pipeline → SQLite DB → Dashboard
```

## 📊 KPIs Tracked
- Monthly Revenue
- Revenue by Country
- Revenue by Category  
- Top 5 Products

## 🛠️ Tech Stack
- Python (ETL)
- SQLite (Database)
- Pandas (Transformation)
- Matplotlib (Dashboard)

## 🚀 How to Run
```bash
# 1. Run the pipeline
python etl/pipeline.py

# 2. Open dashboard
jupyter notebook dashboard.ipynb
```
